# JsLibs

> jsLIbs 里面所有插件无论是从代码的角度，还有 api 的设计现在看起来都不理想，一是没时间再重构，二是目前有比 seajs 更好的解决方案了。没有特殊情况不会再更新

要来就来一包！一包五颜六色的 seajs 插件,满足前端开发的各种需求。有校验插件，分页插件，ajax 插件，弹窗插件等等。。
